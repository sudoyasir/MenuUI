OLEDMenuUI

OLEDMenuUI is a lightweight, customizable, and Arduino-friendly UI library designed for SSD1306 OLED displays. It supports vertical menus, button inputs, buzzer feedback, and is easy to extend across multiple microcontrollers.

‚ú® Features

  • Scroll-based menu with selector
  • Clean font rendering using Adafruit GFX
  • Supports external buzzer beeps
  • Padding, arrow indicators, and item highlighting
  • Designed for 128x32 OLED (but adaptable to others)
